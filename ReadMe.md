### Getting Started

Make sure to install dependencies package.json both in client and server folder


```
client/ npm i
```
```
server/ npm i
```

### Testing Server

Running testing in folder server with
```
npm test
```

### Running the web

First, go to server folder and run:
```
npm run dev
```
Second, go to client folder and run:
```
npm start
```
